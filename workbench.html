Installing Spring Hinges and Pull Strings on a 2m RES Glider
A lightweight method for actuating RC sailplane rudders and elevators is with a single pull string for each control, and with a torsion spring in the control surface. In this video, we show how to make and install the spring, hinge, and pull string. The model is a 2m F3RES Slite. The same technique applies to Hand Launch gliders and other small sailplanes. Presented by John McNeil

Capacity and Wire-size Planning for Radios, Motors, and BECs
